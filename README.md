# Softdev

Three accessibility tools built as part of a software development project.

---

**VisioApp** — A mobile app for visually impaired users. It uses the phone camera to detect nearby objects in real time (YOLOv8), describes the surrounding environment using Gemini AI, and reads everything aloud. Also includes GPS navigation and saved places. Built with React Native / Expo on the front end and a Python FastAPI server on the back end.

**VisioSpeak** — A browser extension that transcribes microphone audio into live captions. Supports emotion tags, keyword highlighting, 11 languages, and adjustable display settings like font size and high contrast. Built as a Manifest V3 extension using the Web Speech API.

**Braille 3D Model Generator** — A web tool that takes plain English text and converts it to a 3D-printable Braille STL file. You can tweak dot size, spacing, and shape, and see a live 3D preview before downloading. No install needed — runs fully in the browser using Three.js.
